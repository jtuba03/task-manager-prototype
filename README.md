# TechStart — Student Task Manager (Prototype)

A simple Python-based task management system designed for students to track their project progress.

## Features
- **Add Tasks:** Create new tasks with custom names and priority levels (High, Medium, Low).
- **View Tasks:** List all current tasks and their statuses.
- **Complete Tasks:** Mark tasks as "Done" by their ID.
- **Task Summary:** (Added) A status-at-a-glance report showing the count of tasks grouped by their current status (e.g., Pending, Done).

## How to Run
```bash
python task_manager.py
```
